Getting started with SPLASH extended

These are the basic functionalities

  1. Login

    Open example splash extended app

    Use the following credentials to test our example app --> username: admin | password: admin

  2. Show and search your customers

    Show all customers

    Search for a customer and click on a list item to edit the customer

  3. Edit customer

    Edit the customer data, save the data and click dashboard in the top menu

  4. Show the dashboard

    Switch to your personal data by clicking the right box

  5. Add or change your personal data

    You can add or change your personal data and save the changes by clicking the save button
