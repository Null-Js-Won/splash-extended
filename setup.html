Setup

Sign up for a Salesforce developer accout

Click the link and sign up as Salesforce developer https://developer.salesforce.com/


Deploying to Heroku using the Heroku Button

You can deploy your own version of SPLASH extended using deploy on heroku button using the button bellow:

  1. Click the button

  2. Heroku sign up or login

    Sign up if you don't have an heroku account or login directly

  3. Deploy to Heroku

    Add an app name (optional), choose your runtime selection and click deploy for free

  4. Connect the Heroku app with your Salesforce developer account

    Connect the Heroku app with your Salesforce developer account by clicking at the Heroku Connect Add-on

    Click the setup connection button

    Click next

    Select Environment and API Version

    Enter your Salesforce developer credentials

  5. Configure the mapping between your Salesforce Instance and Heroku

    Under Settings klick on "Import/Export Configuration"

    Click Import

    Right click and save as to download the contacts-mapping.json file.
    Select the file, you just downloaded and press "Upload"

    Now you can use the SPLASH extended application


Deploying to Heroku using the command line

You can also deploy SPLASH extended on heroku using command line with the steps bellow:

  1. Heroku login (using your Heroku credentials: Email and Password)

    $ heroku login

  2. Clone the repository on GitHub

    $ git clone https://github.com/logiclinegmbh/splash-extended

  3. Create a Heroku application

    $ cd splash-extended
$ heroku create

  4. Install the Heroku Postgress Add-on

    $ heroku addons: add heroku-postgresql:hobby-dev

  5. Install the Heroku Connect Add-on

    $ heroku addons: add herokuconnect:demo

  6. Add buildpacks

    $ heroku buildpacks:add --index 1 https://github.com/heroku/heroku-buildpack-nodejs
$ heroku buildpacks:add --index 2 https://github.com/heroku/heroku-buildpack-java

  7. Deploy to Heroku

    $ git push heroku master

  8. Run the the application

    Open the application in a browser

    $ heroku open

    You can see your application running in your browser

  9. For further reading you can also take a look at heroku devcenter
